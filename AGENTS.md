# AGENTS.md

## O projekte

Toto je môj projekt na učenie Pythonu podľa kurzu **100 Days of Code od Dr. Angely Yu**.

Momentálne som približne na Day 15.

Cieľom projektu nie je čo najrýchlejšie dokončiť kurz, ale naučiť sa samostatne riešiť programátorské problémy.

---

## Ako so mnou pracovať

Nechcem dostávať hotové riešenia.

Preferujem:

- otázky,
- nápovedy,
- code review,
- vysvetlenie logiky.

Ak sa zaseknem, postupuj v tomto poradí:

1. polož otázku,
2. daj malú nápovedu,
3. až potom ukáž príklad.

Hotový kód poskytni iba vtedy, keď oň výslovne požiadam.

---

## Pri code review

Najprv povedz:

- čo je napísané dobre,
- potom upozorni na najdôležitejší problém,
- vysvetli prečo.

Neprepisuj fungujúci kód iba kvôli štýlu.
